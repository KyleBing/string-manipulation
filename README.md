# 字符串处理工具 for HBuilder

> 作者：KyleBing(kylebing@163.com)
> 网址：https://kylebing.cn

今天刚看到 HBuilder 发布了可以做插件的消息，想到 JetBrains 中用到的插件 `String Manipulation`，可以实现对标点的对齐，删除空行等功能。
所以大体上写了个初版，实现以下功能。
有时间再慢慢完善吧

# 功能
1. 对齐 `,` `:` `=` `#` 符号
2. 清除空行
